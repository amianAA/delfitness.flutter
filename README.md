# Delfitness
Repository to Delfitness mobile app 

## Development
The following tools are required
 - Python 3
 - [Lefthook](https://github.com/evilmartians/lefthook)
 - Flutter
 - Dart

### Running the tests
```bash
flutter test
```

### Install Lefthook locally
```bash
lefthook install
```
